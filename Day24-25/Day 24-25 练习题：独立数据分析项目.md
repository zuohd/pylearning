# Day 24-25 练习题：独立数据分析项目
> 要求：独立完成一个完整的数据分析项目
>

---

## 练习 24：项目开发
```plain
独立完成数据分析项目：

Day 24 任务：
1. 选择数据源
   - 使用爬取的豆瓣数据
   - 或使用提供的模拟数据

2. 数据清洗
   - 处理缺失值
   - 处理异常值
   - 数据类型转换

3. 基本分析
   - 数据概览
   - 基本统计
   - 分组统计

Day 25 任务：
4. 可视化
   - 至少4种不同图表
   - 组合成数据看板

5. 报告撰写
   - 分析目标
   - 分析过程
   - 主要发现
   - 结论建议

6. 代码整理
   - 规范命名
   - 添加注释
   - 确保可运行
```

## 练习 25：项目提交
```plain

- 分析方法

提交检查：
- [ ] 代码能正常运行

- [ ] README 说明清楚
```
