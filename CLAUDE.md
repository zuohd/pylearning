# CLAUDE.md

## 项目概述

面向大三学生的 30 天实习准备培训课程（Python + Go），每天半天授课。纯 Markdown 教材与练习，无可执行代码或构建工具。所有内容使用中文。

当前进度：Day 19（爬虫进阶 + 综合项目阶段）。Day 1-18 课件已定稿，Day 19+ 可能仍在编写或调整中。

## 课程阶段

| 阶段 | 天数 | 主题 |
| --- | --- | --- |
| Python 后端基础 | Day 1-15 | Python 语法 → Flask → MySQL/SQLAlchemy → JWT → 项目实战 |
| 爬虫入门 | Day 16-18 | requests + BeautifulSoup、翻页爬取、豆瓣 Top250 |
| 爬虫进阶 + 综合项目 | Day 19-22 | pandas、数据入库、电影数据 API 综合项目 |
| Go 语言 | Day 23-28 | 基础语法 → Gin → GORM → Go 博客项目 |
| 总结 | Day 29-30 | 项目整理 + 回顾 + 自学资料 |

## 目录结构

```
DayN/
├── Day N：标题.md    # 课件
└── exercises.md      # 练习题
```

每个 `DayN/` 目录包含一个课件和一个练习文件。

## 课件格式约定

```markdown
# Day N：标题
> 学习目标：……

---

## 1. 章节标题
（内容，代码块标注语言：python / go / plain / sql）

---

## 今日小结
| 知识点 | 要点 |
| --- | --- |

---

## 课后练习
完成练习（见 [exercises.md](exercises.md)）
```

- 标题格式：`# Day N：标题`（中文冒号）
- 引用块写学习目标
- 章节用 `---` 分隔，编号递增
- 代码块标记语言（`python` / `go` / `plain` / `sql`）
- 结尾：小结表格 + 练习链接

## 练习格式约定

```markdown
# Day N 练习题：标题
> 要求：……

---

## 练习 1：题目名
```plain
题目描述和示例输出
```

提示：
+ 提示内容

---

## 完成检查
+ [ ] 检查项
```

- 题目内容用 `plain` 代码块包裹
- 提示写在代码块外
- 末尾用复选框列出完成检查项
