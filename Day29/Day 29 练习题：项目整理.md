# Day 29 练习题：项目整理
> 要求：整理代码，完善文档
>

---

## 练习 35：代码检查
```plain
检查并整理你的项目：

1. 代码检查
   - 运行代码确保无错误
   - 检查未使用的变量/导入
   - 检查命名规范

2. 文件整理
   - 删除临时文件
   - 确保 .gitignore 正确
   - 检查是否有敏感信息

3. 格式化
   Python: 运行 black 或 flake8
   Go: 运行 go fmt
```

---

## 练习 36：文档完善
```plain
完善项目文档：

1. 编写 README.md
   - 项目介绍
   - 功能列表
   - 安装说明
   - 使用说明
   - 项目结构

2. 添加代码注释
   - 关键函数添加文档字符串
   - 复杂逻辑添加注释

3. 提交到 GitHub
   - git add .
   - git commit -m "完成项目开发"
   - git push
```

---
