# Day 21 练习题：数据统计与分组
> 要求：掌握 groupby 和统计分析
>

---

## 练习 1：基本统计
```plain
使用豆瓣电影数据（或模拟数据），完成：

1. 计算评分的基本统计
   - 平均值、中位数
   - 最大值、最小值
   - 标准差

2. 按评分排序
   - 找出评分最高的10部电影
   - 找出评分最低的10部电影

3. 评分分布
   - 统计每个评分的电影数量
   - 计算评分的四分位数
```

## 练习 2：分组统计
```plain
模拟数据：
data = {
    'city': ['北京', '上海', '北京', '上海', '广州', '广州', '北京'],
    'category': ['电子', '电子', '服装', '服装', '电子', '食品', '食品'],
    'sales': [1000, 1500, 800, 900, 1200, 600, 700],
    'quantity': [10, 15, 20, 18, 12, 30, 25]
}

完成：
1. 按城市分组，计算销售额总和
2. 按城市分组，计算销售额平均值和数量总和
3. 按城市和类别分组，计算销售额总和
4. 找出每个城市销售额最高的类别
```

## 练习 3：透视表
```plain
使用练习 2 的数据，创建透视表：

1. 行：城市
   列：类别
   值：销售额总和

2. 添加行合计和列合计

3. 用 0 填充空值
```
