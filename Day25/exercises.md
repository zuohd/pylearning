## 遇到问题？

```
问题1: 中文显示为方框
解决: 确保设置了中文字体
      plt.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS']

问题2: 图表重叠
解决: 使用 plt.tight_layout()
      或调整 figsize

问题3: 保存的图片是空白的
解决: 确保在 plt.show() 之前调用 plt.savefig()

问题4: 找不到数据文件
解决: 检查工作目录和文件路径
      使用绝对路径或 os.path.join()
```
