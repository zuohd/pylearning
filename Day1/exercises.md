## 练习 1：简单计算器
```plain
编写程序，完成以下任务：

1. 获取用户输入的两个数字
2. 计算并输出它们的：
   - 加法结果
   - 减法结果
   - 乘法结果
   - 除法结果
   - 整除结果
   - 取余结果

示例输出：
请输入第一个数字：10
请输入第二个数字：3
10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3.3333333333333335
10 // 3 = 3
10 % 3 = 1
```

## 练习 2：温度转换
```plain
编写程序，完成以下任务：

1. 获取用户输入的摄氏温度
2. 将摄氏温度转换为华氏温度
3. 公式：华氏 = 摄氏 * 9 / 5 + 32
4. 输出转换结果（保留两位小数）

示例输出：
请输入摄氏温度：36.5
36.5 摄氏度 = 97.70 华氏度

提示：
- 使用 float() 转换输入
- 使用 f-string 格式化输出，{value:.2f} 保留两位小数
```

## 练习 3：个人信息卡
```plain
编写程序，完成以下任务：

1. 创建以下变量：
   - name（姓名，字符串）
   - age（年龄，整数）
   - height（身高，浮点数，单位：米）
   - is_student（是否学生，布尔值）

2. 使用 f-string 输出个人信息卡
3. 计算并输出出生年份（当前年份 - 年龄）

示例输出：
========== 个人信息卡 ==========
姓名：张三
年龄：20
身高：1.75 米
学生：是
出生年份：2005
================================
```
