# Day 17 练习题：翻页爬取 + 数据存储
> 要求：实现翻页爬取并保存数据到 CSV
>

---

## 练习 1：分析翻页规律
```plain
分析以下网站的翻页 URL 规律：

1. http://quotes.toscrape.com/
   观察第1页、第2页、第3页的 URL
   写出 URL 规律

2. 假设某网站列表页 URL 为：
   第1页: https://example.com/list?page=1&size=10
   写出爬取前5页的 URL 列表
```

---

## 练习 2：翻页爬取
```plain
爬取 http://quotes.toscrape.com/ 的前3页数据：

1. 分析翻页 URL 规律
2. 实现翻页爬取
3. 每页爬取后延时 1-2 秒
4. 收集所有名言数据
5. 打印爬取统计信息（共多少页，多少条数据）
```

---

## 练习 3：CSV 存储
```plain
将爬取的数据保存到 CSV 文件：

1. 将练习 2 爬取的数据保存为 quotes.csv
2. CSV 格式：
   - 列名：text, author, tags
   - 编码：utf-8-sig

3. 实现读取功能，读取 CSV 并打印前5条

4. 额外挑战：实现增量爬取
   - 第一次运行爬取并保存
   - 第二次运行只保存新数据
```
