## 练习 1：邮箱正则验证
```plain
实现并测试邮箱验证函数：

1. 编写 validate_email(email) 函数
2. 使用正则表达式验证邮箱格式
3. 集成到注册接口

测试用例：
validate_email("user@example.com")    # None（合法）
validate_email("user@domain.cn")      # None（合法）
validate_email("user.name@mail.com")  # None（合法）
validate_email("invalid")             # "邮箱格式不正确"
validate_email("@domain.com")         # "邮箱格式不正确"
validate_email("user@")               # "邮箱格式不正确"
validate_email("")                    # "邮箱格式不正确"
```

## 练习 2：关键词搜索
```plain
实现完整的用户搜索功能：

1. GET /api/users?keyword=张
   - 搜索用户名或邮箱包含关键词的用户
   - 支持分页

2. GET /api/users?keyword=张&status=1
   - 搜索 + 状态筛选

3. GET /api/stats/users
   - 返回用户总数、今日新增、本周新增

测试步骤：
- 创建 5 个以上测试用户
- 用不同关键词搜索
- 验证分页参数
- 验证统计数据
```
