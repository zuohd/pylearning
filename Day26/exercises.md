> 要求：掌握 Go 语言基础
>

---

## 练习 1：环境和基础
```plain
1. 安装 Go 并验证
   - 运行 go version
   - 创建项目 go mod init hello

2. 编写 Hello World
   - 打印 "Hello, Go!"
   - 编译并运行

3. 变量练习
   - 声明整数变量
   - 声明浮点数变量
   - 声明字符串变量
   - 使用 := 简短声明
   - 声明常量
```

---

## 练习 2：控制流
```plain
1. 条件语句
   写一个函数 checkScore(score int)
   - score >= 90: 打印 "优秀"
   - score >= 80: 打印 "良好"
   - score >= 60: 打印 "及格"
   - 其他: 打印 "不及格"

2. 循环
   - 打印 1 到 10
   - 计算 1 到 100 的和
   - 打印九九乘法表

3. switch 语句
   写一个函数 getDayName(day int) string
   返回星期几的名称
```

---

## 练习 3：切片和 Map
```plain
1. 切片练习
   - 创建一个整数切片，包含 1-5
   - 追加 6, 7, 8
   - 获取切片的前3个元素
   - 获取切片的后3个元素
   - 遍历并打印所有元素

2. Map 练习
   - 创建一个 map，存储学生成绩
     张三: 85, 李四: 90, 王五: 88
   - 添加 赵六: 92
   - 查找 张三 的成绩
   - 检查 钱七 是否存在
   - 删除 李四
   - 遍历并打印所有学生成绩
```
