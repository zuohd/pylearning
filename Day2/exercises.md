## 练习 1：猜数字游戏
```plain
编写一个猜数字程序：

1. 程序预设一个 1-100 之间的数字
2. 用户不断输入猜测值
3. 程序提示"大了"或"小了"
4. 猜对后显示"恭喜你，猜对了！用了 N 次"
5. 输入 quit 可以退出游戏

提示：
- 使用 while True 循环
- 使用 break 退出循环
- 用一个变量记录猜测次数
```

---

## 练习 2：FizzBuzz
```plain
编写 FizzBuzz 程序：

1. 打印 1 到 100 的数字
2. 如果数字能被 3 整除，打印 "Fizz"
3. 如果数字能被 5 整除，打印 "Buzz"
4. 如果同时能被 3 和 5 整除，打印 "FizzBuzz"
5. 否则打印数字本身

示例输出：
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
...
```

## 练习 3：累加器
```plain
编写一个累加器程序：

1. 用户不断输入数字
2. 每次输入后显示当前总和
3. 输入 0 时结束，显示最终总和和输入次数

示例输出：
请输入数字（0 结束）：10
当前总和：10
请输入数字（0 结束）：20
当前总和：30
请输入数字（0 结束）：5
当前总和：35
请输入数字（0 结束）：0
结束！总和：35，共输入 3 次
```

## 练习 4：密码强度检验
```plain
编写密码强度检验程序：

1. 获取用户输入的密码
2. 按以下规则评分（每项 1 分）：
   - 长度 >= 8
   - 包含数字
   - 包含大写字母
   - 包含小写字母
3. 根据总分输出强度：
   - 4 分：强
   - 3 分：中
   - 2 分及以下：弱

提示：
- 使用 for 循环遍历密码字符
- 使用 char.isdigit()、char.isupper()、char.islower()
```
