## 练习 1：处理缺失值
```plain
给定以下数据：

data = {
    'name': ['张三', '李四', None, '赵六', '钱七'],
    'age': [20, None, 22, 20, None],
    'score': [85, 90, np.nan, 92, 78],
    'city': ['北京', '上海', '广州', None, '深圳']
}

完成：
1. 检查每列的缺失值数量
2. 删除包含任何缺失值的行
3. 用均值填充 age 列的缺失值
4. 用 '未知' 填充 city 列的缺失值
5. 用前一个值填充 name 列的缺失值
```

## 练习 2：数据类型和字符串处理
```plain
给定以下数据：

data = {
    'id': ['001', '002', '003', '004'],
    'price': ['100.5', '200.3', '150.8', 'N/A'],
    'date': ['2024-01-15', '2024-01-16', '2024-01-17', '2024-01-18'],
    'name': ['  Apple  ', 'BANANA', 'orange ', 'Grape']
}

完成：
1. 将 id 转换为整数
2. 将 price 转换为浮点数（N/A 变成 NaN）
3. 将 date 转换为日期类型
4. 清理 name 列：去除空白，首字母大写
5. 打印转换后的数据类型
```

## 练习 3：重复值和异常值
```plain
给定以下数据：

data = {
    'name': ['张三', '李四', '张三', '王五', '李四'],
    'score': [85, 150, 85, -10, 90]
}

完成：
1. 检查并打印重复行
2. 删除重复行（保留第一个）
3. 找出 score 的异常值（< 0 或 > 100）
4. 将异常值限制在 0-100 范围内
5. 打印处理后的数据
```
