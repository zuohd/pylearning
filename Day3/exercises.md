## 练习 1：购物清单
```plain
编写一个购物清单程序：

1. 支持以下操作：
   - add <商品名> <价格>：添加商品
   - remove <商品名>：删除商品
   - list：显示所有商品和总价
   - quit：退出程序

2. 使用列表存储商品（每个商品是一个字典）

示例交互：
> add 苹果 5.5
已添加：苹果 ¥5.50
> add 牛奶 8.0
已添加：牛奶 ¥8.00
> list
1. 苹果 - ¥5.50
2. 牛奶 - ¥8.00
总计：¥13.50
> remove 苹果
已删除：苹果
> quit
```

## 练习 2：词频统计
```plain
编写一个词频统计程序：

1. 给定一段文本（可以硬编码）
2. 统计每个单词出现的次数
3. 按出现次数从高到低排序输出

示例输入：
text = "python is great python is easy python is fun"

示例输出：
python: 3
is: 3
great: 1
easy: 1
fun: 1

提示：
- 使用 split() 分割字符串
- 使用字典存储词频
```
